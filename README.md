# Generative Tag

Game you played as a kid... but on yout phone

## Used technologies

- [P5.js](https://p5js.org/)
- [socket.io](https://socket.io/)
- [node.js](https://nodejs.org/en/)

## Development

Run from root, unless told otherwise.

### Run frontend

```
npx live-server
```

### Run backend

```
cd /backend

npx nodemon server.js
```
